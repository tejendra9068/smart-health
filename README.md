## 📖 Overview

**Smart Health Network Monitor** is a modern healthcare management platform designed to help hospitals, Primary Health Centers (PHCs), Community Health Centers (CHCs), and healthcare administrators monitor facilities, manage medicine inventory, track patient records, and receive real-time alerts from a centralized dashboard.

The system provides an intuitive interface for healthcare professionals to manage multiple facilities efficiently while ensuring medicine availability, patient record management, and operational transparency.

The application features both a **Healthcare Dashboard** and an **Admin Command Center** for complete network management.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Smart-Health-Network-Monitor.git
```

Move into project

```bash
cd Smart-Health-Network-Monitor
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

## 📢 About the Project

**Smart Health Network Monitor** is a comprehensive healthcare operations platform that digitizes healthcare facility management through centralized dashboards, inventory monitoring, analytics, patient registration, and intelligent alert management. It empowers healthcare administrators to oversee multiple facilities efficiently, monitor medicine stock levels, optimize bed utilization, and respond quickly to critical healthcare events. The platform combines a clean, modern user interface with real-time operational insights to improve healthcare service delivery and resource management.
